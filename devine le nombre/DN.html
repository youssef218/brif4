Enonce

L’utilisateur doit deviner un nombre entre 1 et 50 que le programme a choisi aléatoirement en 10 tentatives.
Quand l’utilisateur entre une valeur, on lui affiche le nombre de tentatives restantes, ainsi qu’un message indiquant si sa réponse est correcte ou pas.

Démo

Code source

let tentative=1; let randomNr= Math.floor(Math.random()*50)+1; document.getElementById('Envoyer').onclick=function(){ document.getElementById('tr').innerHTML="tentatives restantes"+tentative+"/10" let guess = document.getElementById('nombr1').value; if (randomNr==guess && tentative<=10){ document.getElementById('expression').innerHTML="BRAVO"; tentative++; } else if( randomNr!=guess && tentative<10){ document.getElementById('expression').innerHTML="Réesseyez encore !"; tentative++; } else{ document.getElementById('expression').innerHTML="pas "; } }
