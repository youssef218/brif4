Enonce

Ecrire un programme Javascript qui demande à l’utilisateur de saisir une phrase dans une zone
de texte et un mot dans une autre zone de texte. Le clique sur le bouton rechercher affiche
un message si le mot existe dans la phrase ou non. Le clique sur le bouton supprimer permet
d’effacer toutes les occurrence du mot dans la phrase ( si le mot existe bien sûr).

Démo


Le message de recherche s’affiche ici Ou la nouvelle phrase après la suppression des occurences

Code source

document.getElementById('chercher').onclick=function(){ let caros=document.getElementById('enter').value ; let objt=document.getElementById('obj').value; if(caros.includes(objt)){ document.getElementById('ed').innerHTML="le mot existe dans la phrase" } else{ document.getElementById('ed').innerHTML="le mot ne existe pas dans la phrase" } } document.getElementById('Supprimer').onclick=function(){ let caros=document.getElementById('enter').value ; let objt=document.getElementById('obj').value; document.getElementById('ed').innerHTML=caros.replace(objt,""); }
